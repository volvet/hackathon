Performance Report Protocol
Version: 0.1  


0. REPORT FORMAT(HTTP POST REQUEST BODY)
SESSION:   SESSION_ID
COMMMAND:  COMMAND_ID
COMMAND_SPECIFIC_ITEMS:   

1.  Performance Report  
   SESSION:  PERFORMANCE

(1) Start Session
   COMMAND:  START-SESSION
   TIME:     MM-DD-YYYY-HH-MM-SS
   DEVICE:   XXXXXX
   OS:       iOS/Android
   CPU:      XXXXXX
   MEMORY:   XXXXXX
   
 (2) Start Call
   COMMAND:  START-CALL
   TIME:     MM-DD-YYYY-HH-MM-SS
   VIDEO:    ON/OFF
   if( VIDEO == ON )
     RESOLUTION:   WWW-HHH
     FPS:          RRRR
   AUDIO:    ON/OFF

  (3) Stat Recv
    COMMAND:  STAT-RECV
    TIME:     MM-DD-YYYY-HH-MM-SS
    VIDEO:    ON/OFF
    if( VIDEO == ON )
       RESOLUTION: WWW-HHH
       FPS:        RRRR
    AUDIO:    ON/OFF
   
  (4) INFO
    COMMAND:   INFO
    TIME:     MM-DD-YYYY-HH-MM-SS
    BATTERY:   XX
    CHARGING:  YES/NO
    CPU:       XX
    
  (5) Stop Call
    COMMAND:  STOP-CALL
    TIME:     MM-DD-YYYY-HH-MM-SS
    
  (6) Stop Session
    COMAND:   STOP-SESSION
    TIME:     MM-DD-YYYY-HH-MM-SS
